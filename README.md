# college_event_management
College Event Management System
Description
This is a web application developed using html,css,js,Java(servlet),JSP,andMySQL.

Features
admin create dept,view reports,view depts
event organizer create event,edit,delete events
event organizer accpet or reject event request
Student login/sign in
student register self info form
register for events send request to event organizer
student see approve reject req
pay events fee
download pass
attendance system only one time
Technologies Used
Java (Servlet, JSP)
HTML, CSS, JavaScript
MySQL
NetBeans
Database Setup
Open MySQL Workbench
Create a new database
Import the college_event.sql file
Run the project
How to Run
Import project in NetBeans
Configure database
Run on Apache Tomcat
